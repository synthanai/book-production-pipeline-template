# archive/

**Purpose:** Archive for old audio scripts, narration versions, and recordings

---

## What Goes Here

- Previous narration script versions
- Old chapter recordings (before retakes)
- Superseded narration guides
- Previous audiobook compilations
- Alternate narration approaches

## When to Archive

- Before script revisions
- After recording retakes
- When narration approach changes
- Before final audiobook compilation
