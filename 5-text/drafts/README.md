## ğŸ”„ Draft Workflow

### Writing Process
1. Start new chapter in drafts/
2. Experiment freely
3. Try different approaches
4. Don't worry about perfection

### Promotion to Chapters
1. When draft is substantial and validated
2. Move/copy to ../chapters/
3. Keep draft as archive if desired
4. Or delete if superseded

### Managing Drafts
- Archive old drafts when new version created
- Delete clearly failed experiments
- Keep alternatives for reference
- Move to ../../archive/ periodically

## ğŸ¯ Best Practices

### Freedom to Experiment
- This is your safe space
- Try bold ideas
- Write without self-editing
- Explore alternatives

### Organization
- Use clear naming conventions
- Date significant versions
- Note what makes alternatives different
- Clean up regularly

### Know When to Promote
- Draft doesn't need to be perfect
- Just needs to be substantial and directionally correct
- Can refine further in chapters/
