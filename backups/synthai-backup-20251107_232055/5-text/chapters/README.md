## ğŸ“ File Organization

### Naming Convention
```
Chapter-[XX]-[Title-Slug].md
```

- Use two-digit chapter numbers (01, 02, ..., 21)
- Use descriptive title slugs
- Use hyphens, not spaces
- Markdown format (.md)

### Examples
- `Chapter-01-Awakening.md`
- `Chapter-07-The-Crisis.md`
- `Chapter-15-Breakthrough.md`
- `Chapter-21-Resolution.md`
