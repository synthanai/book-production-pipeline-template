## üìù Snippet Organization

### By Character
```markdown
# Character Snippets: [Character Name]

## Physical Description
[Standard description of appearance]

## Typical Mannerisms
- [Mannerism 1]
- [Mannerism 2]

## Speech Pattern
[How they talk, typical phrases]

## Emotional States
**Angry:** [Description when angry]
**Happy:** [Description when happy]
```

### By Location
```markdown
# Location Snippets: [Location Name]

## First Appearance Description
[Full description for first mention]

## Brief Reference
[Shortened version for subsequent mentions]

## Sensory Details
- **Visual:** [What it looks like]
- **Auditory:** [What sounds are present]
- **Olfactory:** [What smells]
```

### By Type
```markdown
# Transition Snippets

## Time Transitions
- "Three days later..."
- "The following morning..."
- "Hours passed before..."

## Location Transitions
- "Across town, ..."
- "Meanwhile, at..."
- "Back at..."

## POV Transitions
- "[Character]'s perspective shifted when..."
```

## üéØ When to Create Snippets

### Create snippet if:
- Description appears 3+ times
- Consistency is critical
- Complex description is hard to recreate
- Standard phrasing exists
- Quick insertion would save time

### Don't snippet:
- Unique, one-time descriptions
- Highly contextual content
- Creative variations desired
- Simple, obvious phrases

## üìñ Using Snippets Effectively

### During Writing
1. Reference snippet library
2. Copy relevant snippet
3. Adapt for specific context
4. Maintain core consistency

### During Editing
1. Find inconsistent descriptions
2. Create or update snippets
3. Standardize across manuscript
4. Remove variations

### Building Library
1. Extract from existing writing
2. Create during prep phase
3. Add as patterns emerge
4. Organize logically

---

## üîß Snippet Maintenance

### Keep snippets:
- **Current** - Update when descriptions evolve
- **Organized** - Easy to find what you need
- **Versioned** - Track significant changes
- **Documented** - Note where each is used

### Review regularly:
- Remove unused snippets
- Consolidate duplicates
- Update outdated descriptions
- Add new patterns

## üí° Snippet vs. Reference

| Aspect | Snippet | Reference |
|--------|---------|-----------|
| **Purpose** | Text to copy | Facts to look up |
| **Format** | Prose/dialogue | Tables/lists |
| **Use** | Insert into writing | Check for accuracy |
| **Length** | Sentences/paragraphs | Single facts |
