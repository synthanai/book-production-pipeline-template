## ğŸ“Š Characteristics of Master Documents

### Comprehensive
- Contains ALL information on its topic
- 50,000 - 200,000+ words per document
- Extremely detailed
- Authoritative reference

### Authoritative
- Highest tier in document hierarchy
- Resolves conflicts between other docs
- Updated carefully and deliberately
- Versioned and tracked

### Foundational
- Created early in prep phase
- Referenced by all other documents
- Guides all subsequent writing
- Rarely contradicted

## ğŸ”„ Update Protocol

Master documents should be updated:
- During prep phase as you refine concepts
- When major story/character changes occur
- After discovering inconsistencies
- At major project milestones

**Always:**
- Version the document
- Note what changed in header
- Update "Last Modified" date
- Validate against other masters
