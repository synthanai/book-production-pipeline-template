## ğŸ“– Reference Format Template

### Table Format (Recommended)
```markdown
# Reference: Character Quick Sheet

| Character | Age | Role | Key Trait | Speech Pattern |
|-----------|-----|------|-----------|----------------|
| Maya      | 34  | Leader | Pragmatic | Direct, concise |
| Arjun     | 29  | Specialist | Idealistic| Thoughtful, measured |
```

### List Format
```markdown
# Reference: Framework Terms

## Framework Name
- **Definition:** Core system or methodology
- **Components:** Key elements
- **Application:** How it's used

## Element One
- **Definition:** First element...
- **Key Principle:** ...
```

### Quick Facts Format
```markdown
# Reference: Location - Market District

**Area:** Central City
**Atmosphere:** Busy commercial district
**Key Details:** Shops, markets, cultural landmarks
**Story Use:** Chapters 4, 9, 14
```

## ğŸ¯ Best Practices

1. **Keep concise** - Resist elaboration
2. **Use tables** - Easier to scan than prose
3. **Alphabetize** - Enable quick lookup
4. **Cross-reference** - Link to source documents
5. **Update regularly** - Keep in sync with masters
6. **One per topic** - Don't combine unrelated references
