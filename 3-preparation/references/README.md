## üìñ Reference Format Template

### Table Format (Recommended)
```markdown
# Reference: Character Quick Sheet

| Character | Age | Role | Key Trait | Speech Pattern |
|-----------|-----|------|-----------|----------------|
| Maya      | 34  | CEO  | Pragmatic | Direct, concise |
| Arjun     | 29  | Dev  | Idealistic| Thoughtful, measured |
```

### List Format
```markdown
# Reference: Framework Terms

## SYNTHAI
- **Definition:** System for organizational transformation
- **Components:** 5 practices
- **Level:** Organizational consciousness

## SUPPORT
- **Definition:** First practice...
- **Key Principle:** ...
```

### Quick Facts Format
```markdown
# Reference: Location - T. Nagar

**Neighborhood:** Central Chennai
**Atmosphere:** Busy commercial district
**Key Details:** Jewelry shops, Pondy Bazaar
**Story Use:** Chapters 4, 9, 14
```

## üéØ Best Practices

1. **Keep concise** - Resist elaboration
2. **Use tables** - Easier to scan than prose
3. **Alphabetize** - Enable quick lookup
4. **Cross-reference** - Link to source documents
5. **Update regularly** - Keep in sync with masters
6. **One per topic** - Don't combine unrelated references

## üí° Reference vs. Guide vs. Master

| Aspect | Reference | Guide | Master |
|--------|-----------|-------|--------|
| **Purpose** | Quick lookup | How-to instructions | Comprehensive authority |
| **Length** | 2-10K words | 20-90K words | 50-200K words |
| **Format** | Tables, lists | Detailed prose | Comprehensive prose |
| **Use** | During writing | Before/during writing | During prep, major decisions |
| **Detail** | Essential only | Detailed instructions | Everything |
